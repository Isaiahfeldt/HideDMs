# HideDMs (Vencord Plugin)


**Hide specific DM threads** via right‑click. Hidden DMs disappear entirely from the UI and produce **no notifications or sounds** until revealed or unhidden.


## Features
- Right‑click any DM → **Hide/Unhide**
- **Persistent** hidden list across restarts
- **Secret reveal** hotkey (Ctrl+Shift+<Key>, default `K`) to temporarily show all hidden DMs (configurable in plugin settings)
